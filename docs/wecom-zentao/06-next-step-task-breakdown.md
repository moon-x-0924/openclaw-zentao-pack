# 企微对话式禅道助手研发拆解

## 产品任务

- 明确 MVP 范围和优先级
- 确认高风险动作清单
- 确认身份映射策略和绑定流程

## 前端 / 客户端任务

- 输出企微消息卡片规范
- 设计自建应用草案确认页
- 设计复杂表单承接页

## 后端任务

- 回调接入与去重
- 会话上下文管理
- 权限和身份映射校验
- 禅道执行能力统一封装

## 集成任务

- 企业微信回调联调
- 禅道权限与账号映射联调
- OpenClaw 路由与技能包接入联调

## 测试任务

- 高频查询回归
- 创建 / 流转回归
- 权限边界测试
- 高风险动作确认链路测试
- 接口异常和超时恢复测试

## 发布与推广任务

- 明确试点用户和试点群
- 建立使用反馈收集机制
- 准备 FAQ 和常见问题排查说明

## 建议顺序

1. 身份映射与回调
2. 高频查询
3. 核心创建与流转
4. 风险确认与审计
5. 自建应用承接复杂场景
