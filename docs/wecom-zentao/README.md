# 企微对话式禅道助手文档索引

本目录用于承接“企微机器人/自建应用对话式操作禅道”的完整方案、原型和流程文档。它不只是文档流转目录，现在也包含：

- 方案文档
- 产品 PRD
- UI 设计稿草案
- HTML 原型
- 独立泳道图
- 文字版流程说明

如果你第一次进入这个目录，建议优先看：

1. `05-master-solution.md`
2. `09-wecom-ui-prototype.html`
3. `10-cross-role-swimlane.html`

## 文档总览

### 方案基线

- `00-project-brief.md`
  - 项目简述、业务目标、约束、风险、待确认问题
  - 适合快速理解项目背景

- `01-product-plan.md`
  - 产品方案、角色、能力地图、MVP、路线图
  - 适合产品和业务评审

- `02-workflow-design.md`
  - 会话工作流、意图、槽位、补问、确认逻辑
  - 适合机器人工作流设计和 AI/后端联调

- `03-conversation-ui.md`
  - 企微消息 UI、卡片类型、状态设计、交互规范
  - 适合 UI 设计和前端实现参考

- `04-architecture.md`
  - 系统架构、身份映射、授权、审计、幂等、异常处理
  - 适合后端、架构、联调评审

- `05-master-solution.md`
  - 完整总方案
  - 如果只看一份方案文档，优先看这份

- `06-next-step-task-breakdown.md`
  - 面向研发执行的拆解
  - 适合排期、拆任务、做实施计划

### 产品与设计补充

- `07-product-prd.md`
  - 更偏产品 PRD 表达的版本
  - 适合产品、设计、研发一起评审功能边界

- `08-ui-design-draft.md`
  - UI 设计草案和 Figma 标注说明
  - 适合设计师出图和开发理解页面结构

### 可视化原型

- `09-wecom-ui-prototype.html`
  - 主原型页
  - 支持：
    - 按阶段展示
    - 按角色展示
    - 角色子场景切换
    - 跨角色协作视图
  - 适合评审“最终交互长什么样”

- `10-cross-role-swimlane.html`
  - 独立的跨角色协作泳道图
  - 已映射 SOP 阶段与 Step
  - 适合单独发给别人、截图、投屏、开会讲解协作链路

- `10-cross-role-swimlane.md`
  - 独立泳道图的 Markdown 文字版
  - 适合留档、写评审纪要、做实施说明
